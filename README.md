## 本地预览

文档使用 [Docsify](https://docsify.js.org/) 构建。在 `docs` 目录下启动本地服务器即可预览：

```bash
cd docs
npx serve .
# 或
python -m http.server 3000
```

然后访问 `http://localhost:3000`。
