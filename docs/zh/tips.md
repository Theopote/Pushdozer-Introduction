# 使用技巧

## 基础操作

### 操作顺序

1. 选择工作模式
2. 选择笔刷几何体并调整尺寸
3. 设置标高模式
4. 调整模式专属参数（如平滑强度、种植密度等）
5. 确认显示模式与操作距离
6. 右键执行

### 撤销与重做

- 每名玩家独立保存最多 **30 步**操作历史
- `Ctrl+Z` 撤销，`Ctrl+Y` 重做
- 撤销/重做在多人游戏中同步到所有客户端
- 大范围操作前建议先在小区域测试

## 地形塑造工作流

1. **粗加工**：用大尺寸长方体或球体快速塑形
2. **精修**：缩小笔刷，用平滑模式修整细节
3. **自然化**：表面粗糙模式添加微起伏
4. **生态恢复**：批量种植或水岸处理添加植被

## 建筑辅助

- **锁定标高 + 长方体**：创建完美水平地基
- **表层转换**：混合材质地表（如石路与草地交界）
- **分层挖掘**：逐层向下开挖，避免一次破坏过多方块

## 性能优化

- 笔刷尺寸从 1–64 格，建议从较小值开始
- 优先使用 **线框** 显示模式；需要更直观的体积预览时可改用 **表面**
- 大范围操作分批执行，避免一次性影响数万方块
- 多人服务器上操作前通知其他玩家

## 多人协作

- 每人配置独立，互不影响
- 地形修改实时同步给所有在线玩家
- 撤销仅影响自己的操作历史
- 大范围操作前与队友沟通，避免冲突

## 配置管理

- 配置保存在 `config/pushdozer_config.json`
- 建议通过游戏内界面修改，而非直接编辑 JSON
- 旧版 `pushdozer.json` 会在首次加载时自动迁移
