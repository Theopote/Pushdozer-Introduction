# 常见问题

## 安装问题

### 安装后无法启动游戏

请检查：

1. Fabric API 是否已正确安装且版本匹配 1.21.11
2. Fabric Loader 版本是否 ≥ 0.18.2
3. Java 版本是否为 21
4. 是否有其他模组冲突

### 工具不显示在物品栏中

请确认：

1. 处于创造模式
2. 在 **Tools（工具）** 分组中查找
3. 模组已正确加载（检查模组列表）
4. 没有其他模组覆盖物品栏分组

## 使用问题

### 操作没有反应

可能原因：

1. 未手持推土机工具（主手或副手均可）
2. 工作模式或笔刷配置有误
3. 目标位置超出最大操作距离
4. 笔刷尺寸超出允许范围（1–64）
5. 标高限制阻止了操作（如锁定标高下无法破坏更低方块）

### 快捷键不生效

1. 确认手持推土机工具
2. 检查 **选项 → 控制 → 推土机** 中是否有按键冲突
3. 撤销/重做需要同时按住 `Ctrl` 键

### 笔刷尺寸被拒绝

服务端会检查笔刷尺寸是否在 1–64 范围内。如果提示超出范围，请在配置界面中调小笔刷参数。

### 种植/树木生成失败

1. 检查是否开启了 **遵守生物群系规则**，当前生物群系可能不适合所选植物
2. 尝试切换为 **生物群系自适应** 树种/花组
3. 自定义植物列表为空时会使用默认植物

## 性能问题

### 游戏卡顿

1. 减小笔刷尺寸
2. 切换到 **不显示** 预览模式
3. 降低最大操作距离
4. 分批执行大范围操作

### 多人服务器延迟

1. 避免多名玩家同时进行大范围操作
2. 将笔刷尺寸控制在合理范围
3. 与管理员沟通服务器性能状况

## 配置问题

### 配置文件在哪里？

`config/pushdozer_config.json`（文件名始终为此；旧版扁平 JSON 结构会在加载时自动迁移）

### 可以直接编辑 JSON 吗？

可以，但不推荐。建议通过游戏内配置界面修改，以避免格式错误。模组支持从旧版扁平 JSON 格式自动迁移。

## 多人游戏

### 其他玩家看不到我的修改？

1. 检查网络连接
2. 尝试重新登录
3. 查看服务器日志是否有网络错误

### 配置会同步到其他玩家吗？

不会。配置会同步到**服务端**供你自己的操作使用，但**不会广播给其他玩家**，也不会写入他人的本地文件。每名玩家的工作模式、笔刷形状等设置互不影响。

### 所有人都能使用推土机吗？

是的，Pushdozer 默认对所有玩家开放。服务器管理员可通过区域保护插件限制特定区域。
