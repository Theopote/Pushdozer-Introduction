# 入门指南

## 安装要求

| 组件 | 版本要求 |
|------|----------|
| Minecraft | 1.21.11 |
| Fabric Loader | ≥ 0.18.2 |
| Fabric API | 对应 1.21.11 版本 |
| Java | 21（开发与运行均需 JDK 21） |
| 可选 | ModMenu（在模组列表中快速跳转链接） |

### 安装步骤

1. 安装 [Fabric 模组加载器](https://fabricmc.net/use/)
2. 下载并安装 [Fabric API](https://modrinth.com/mod/fabric-api)
3. 从 [GitHub Releases](https://github.com/theopote/pushdozer/releases) 或 [Modrinth](https://modrinth.com/mod/pushdozer) 下载 Pushdozer
4. 将 JAR 文件放入 `.minecraft/mods` 文件夹
5. 启动游戏（客户端或专用服务器均可）

## 首次使用

1. 进入创造模式，在物品栏 **Tools（工具）** 分组中找到 **推土机**
2. 将推土机放入快捷栏并手持
3. 按 `K` 键打开配置界面（可在 **选项 → 控制 → 推土机** 中修改快捷键）
4. 在配置界面中选择工作模式、笔刷形状和参数
5. **右键点击**目标位置执行操作

## 基础快捷键

所有快捷键均需在手持推土机时生效，可在 Minecraft 控制设置中自定义。

| 按键 | 功能 |
|------|------|
| `K` | 打开/关闭配置界面 |
| `G` | 循环切换工作模式 |
| `U` | 循环切换笔刷几何体（8 种） |
| `V` | 循环切换显示模式（线框 → 点云 → 表面 → 不显示） |
| `Ctrl+Z` | 撤销上一步操作 |
| `Ctrl+Y` | 重做操作 |
| `↑` / `↓` | 增加 / 减少最大操作距离 |

> **提示**：右键是执行操作的唯一方式。快捷键仅用于切换设置，不会直接触发地形修改。

## 5 分钟快速上手

### 平整地面

1. 工作模式选择 **挖掘模式**
2. 笔刷选择 **长方体**，调整长、宽、高
3. 标高模式选择 **锁定玩家标高**
4. 右键点击地面

### 铺设地表

1. 工作模式选择 **铺设模式**
2. 铺设方式选择 **自适应生物群系**（或指定自然方块）
3. 右键点击目标区域

### 种植森林

1. 工作模式选择 **批量种植**
2. 种植类型选择 **树木**，树种可选生物群系自适应
3. 调整种植密度与团簇大小
4. 右键点击目标区域

### 创建沙滩

1. 工作模式选择 **水岸处理**
2. 水岸类型选择 **沙滩**
3. 调整水岸宽度，可选启用植被种植
4. 右键点击水域边缘
