# 显示模式

Pushdozer 提供 3 种笔刷预览显示模式，按 `V` 键循环切换：**线框 → 点云 → 不显示**。

预览仅在客户端显示，不影响实际操作范围。

## 线框（Wireframe）

- 以线框轮廓显示笔刷范围边界
- 性能消耗低，适合大多数场景
- 便于精确对齐建筑和判断操作范围
- **推荐作为默认模式**

## 点云（Point Cloud）

- 以点云形式显示笔刷范围内的表面方块
- 比线框更直观地展示影响区域
- 性能消耗略高于线框
- 适合预览铺设、平滑等表面处理操作的效果

## 不显示（None）

- 不显示任何预览
- 性能最优
- 适合熟练用户快速操作
- 录制视频时可减少视觉干扰

## 操作距离

最大操作距离决定你能对多远的位置执行操作（默认 20，最大 99）。

- 按 `↑` 增加距离
- 按 `↓` 减少距离
- 也可在配置界面的滑动条中调整

> **提示**：距离越大，能触及的范围越远，但不会影响笔刷本身的大小。建议根据场景需要调整，避免误操作远处地形。
