# Pushdozer 简介

> Pushdozer 是一个操作简单、功能强大的 Minecraft 地形编辑模组，为建筑师和创作者提供直观的大规模地形修改工具。无论是平整山地、挖掘隧道、批量种植植被，还是处理水岸与表层转换，Pushdozer 都能让这些工作变得轻而易举。

**当前文档版本：v1.1.1** · Minecraft 1.21.11 · Fabric

## 核心特性

- 🚀 **高效地形编辑** — 服务端权威执行，支持批处理与多人同步
- 🎨 **8 种工作模式** — 挖掘、铺设、平滑、表面粗糙、表层转换、撒骨粉、批量种植、水岸处理
- 📐 **8 种笔刷几何体** — 球体、长方体、正八面体、圆柱体、圆锥体、椭球体、正四面体、三棱柱
- 📏 **4 套标高系统** — 标高不限、跟随玩家、锁定玩家标高、自定义标高
- ⚡ **3 种预览模式** — 线框、点云、不显示
- 🔍 **方块过滤** — 精确选择允许破坏的方块类型
- 🔄 **撤销/重做** — 每名玩家 30 步操作历史
- 🌐 **多人联机** — 地形操作实时同步，每人独立配置
