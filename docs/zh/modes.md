# 工作模式

Pushdozer 提供 8 种可通过 `G` 键循环切换的工作模式。每种模式都有独立的配置面板，可在配置界面中进一步调整参数。

## 挖掘模式（EXCAVATE）

清除笔刷范围内的方块。

- **右键点击**执行挖掘
- 通过 **选择允许被破坏的方块** 按钮配置方块白名单
- 支持 **分层挖掘**：按层逐步向下挖掘，可设置挖掘层数
- 配合标高系统可限制挖掘高度范围

**适用场景**：平整地形、挖掘隧道、清除特定方块、创建地下空间。

## 铺设模式（PLACE）

在笔刷范围内铺设地表方块。

- **自适应生物群系**：根据当前生物群系自动选择合适的自然方块（如草原铺草方块、沙漠铺沙子）
- **选择自然方块**：手动指定一种自然方块进行铺设
- 配合 **锁定玩家标高** 可创建水平地面

**适用场景**：填补空地、创建平台、恢复被破坏的地表。

## 平滑模式（SMOOTH）

平滑笔刷范围内的地形起伏。

在配置面板中可选择三种平滑变体：

| 变体 | 说明 |
|------|------|
| 自适应平滑 | 根据周围地形自动判断提升或降低 |
| 平滑提升 | 将地形向上平滑抬升 |
| 平滑降低 | 将地形向下平滑降低 |

- **平滑强度**：控制每次操作的平滑力度（0.0–1.0）
- **平滑强度（重塑）**：0 = 保留原始形状，1 = 完全重塑

**适用场景**：柔化山坡过渡、修整建筑地基边缘、自然化人工地形。

## 表面粗糙（SURFACE_ROUGHEN）

为平整的地表添加自然的起伏纹理。

- **粗糙强度**：控制起伏幅度
- **噪声参数**：频率、持久性、八度数（控制纹理细节）
- **自动缩放**：根据笔刷半径自动调整噪声参数

**适用场景**：让过于平整的人工地形看起来更自然、创建微起伏地貌。

## 表层转换（SURFACE_CONVERT）

将笔刷范围内的地表方块按百分比替换为多种方块。

- 可添加多种方块并设置各自占比
- 支持 **均匀分配** 按钮快速均分比例
- 默认替换为草方块（100%）

**适用场景**：创建混合地表（如草地与泥土斑驳）、地表材质过渡。

## 撒骨粉（BONE_MEAL）

在笔刷范围内对植物使用骨粉效果，促进生长。

- 无需额外配置参数
- 对草、花、作物、树苗等有效

**适用场景**：快速绿化、催熟作物、恢复植被。

## 批量种植（BATCH_PLANT）

在笔刷范围内批量种植植物。

| 种植类型 | 说明 |
|----------|------|
| 树木 | 可选树种（橡树、云杉、白桦等）或生物群系自适应 |
| 花朵 | 可选花组（平原花卉、森林花卉等）或生物群系自适应 |
| 草类 | 种植草、蕨类等低矮植被 |
| 自定义 | 从植物列表中自由选择 |

其他参数：

- **种植密度**：控制单位面积内的植物数量
- **团簇大小**：控制植物分布的聚集程度
- **遵守生物群系规则**：开启后仅在适合的生物群系中种植

**适用场景**：快速造林、布置花田、恢复生态。

## 水岸处理（SHORELINE_PROCESS）

在水域边缘创建自然的水岸过渡效果。

| 水岸类型 | 说明 |
|----------|------|
| 沙滩 | 铺设沙子，适合海洋和湖泊 |
| 堤岸 | 铺设泥土/草方块过渡 |
| 自适应 | 根据生物群系自动选择（如沙漠红沙、雪地雪块） |
| 泥泞 | 泥泞的沼泽风格水岸 |
| 岩石 | 岩石质水岸 |
| 自定义 | 自选方块与植物 |

其他参数：

- **水岸宽度**：过渡区域的宽度，越远替换概率越低
- **启用植物种植** / **植被密度**：在水岸区域自动种植适合的植物
- **在标高以上/以下操作**：限制水岸处理的高度范围

**适用场景**：创建沙滩、河岸边坡、湖泊生态、沼泽水岸。
