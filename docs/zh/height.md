# 标高系统

标高系统控制 Pushdozer 操作时的垂直高度限制，在配置界面的 **标高配置** 面板中设置。

## 四种标高模式

### 标高不限（NO_LIMIT）

- 不限制操作高度
- 可破坏或放置任意 Y 坐标处的方块
- 适合复杂的三维地形编辑

### 跟随玩家标高（FOLLOW_PLAYER）

- 操作标高随玩家移动而更新
- 始终基于玩家当前脚下的 Y 坐标
- 适合边走边编辑的场景

### 锁定玩家标高（LOCKED_ONCE）

- 将标高锁定到切换模式时玩家脚下的 Y 坐标
- 锁定后标高不再随玩家移动
- 挖掘模式：无法破坏低于锁定标高的方块
- 铺设模式：方块放置在锁定标高及以下
- 适合创建精确的水平面

### 自定义标高（CUSTOM）

- 通过滑动条手动设置目标 Y 坐标
- 可精确指定任意高度
- 适合需要固定标高的建筑项目

## 水岸处理的标高限制

水岸处理模式额外提供两个标高开关：

- **在标高以上操作**：所有水岸处理仅在当前标高以上生效
- **在标高以下操作**：所有水岸处理仅在当前标高以下生效

两者互斥，用于精确控制水岸效果的作用高度层。

## 使用技巧

| 场景 | 推荐模式 |
|------|----------|
| 创建水平地面 | 锁定玩家标高 |
| 挖掘地下空间 | 标高不限 |
| 沿河岸逐步推进 | 跟随玩家标高 |
| 在特定楼层施工 | 自定义标高 |
| 水岸仅影响水面附近 | 水岸标高限制 |
