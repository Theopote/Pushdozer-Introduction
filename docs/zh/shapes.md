# 笔刷几何体

Pushdozer 提供 8 种笔刷几何体，按 `U` 键可循环切换。每种几何体有独立的尺寸参数，范围均为 **1–64 格**。

在配置界面中点击 **笔刷形状** 按钮可直接选择任意几何体并调整参数。

## 球体（Sphere）

- **参数**：半径
- **特点**：各向均匀的球形范围
- **适用**：创建山丘、洞穴、平滑过渡、批量种植

## 长方体（Box）

- **参数**：长度、宽度、高度
- **特点**：规则的矩形范围
- **适用**：挖掘地基、创建平台、平整矩形区域、隧道

## 正八面体（Octahedron）

- **参数**：半径
- **特点**：钻石形的立体范围，上下对称
- **适用**：创建尖峰地形、特殊建筑造型

## 圆柱体（Cylinder）

- **参数**：半径、高度
- **特点**：垂直圆柱范围
- **适用**：挖掘竖井、创建塔基、柱状空间

## 圆锥体（Cone）

- **参数**：半径、高度
- **特点**：底部宽、顶部窄的锥形范围
- **适用**：创建火山锥、金字塔形地形

## 椭球体（Ellipsoid）

- **参数**：半径、高度
- **特点**：可拉伸的椭球形范围，水平与垂直比例不同
- **适用**：创建扁平或高耸的自然山丘

## 正四面体（Tetrahedron）

- **参数**：边长
- **特点**：金字塔形的四面体范围
- **适用**：特殊几何造型、尖顶地形

## 三棱柱（Triangular Prism）

- **参数**：边长、高度
- **特点**：三角形截面的柱体范围
- **适用**：楔形地形、三角基座

## 选择建议

| 需求 | 推荐几何体 |
|------|-----------|
| 快速平整大面积 | 长方体 |
| 自然山丘/洞穴 | 球体、椭球体 |
| 竖向挖掘 | 圆柱体 |
| 尖顶山峰 | 圆锥体、正四面体 |
| 规则建筑基础 | 长方体、三棱柱 |
| 植被种植 | 球体（自然分布） |

> **性能提示**：笔刷尺寸越大，单次操作影响的方块越多。建议从小尺寸开始，逐步增大。服务端对笔刷尺寸有上限检查，超出范围会提示调整。
